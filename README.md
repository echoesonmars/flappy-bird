# Flappy Oracle (NIS Programming)

Игровой прототип на `pygame`, вдохновлённый Flappy Bird, с техно-мистическим вайбом и интеграцией OpenAI как Оракула.

## Возможности

- Классический флаппи-геймплей: полёт, трубы, очки.
- Валюта за полёт, покупка скинов (blue/red/yellow).
- Экран настроек: громкость музыки/SFX, ввод OpenAI API ключа, сброс прогресса.
- Оракул:
  - Без ключа — локальные атмосферные ответы.
  - С ключом — ответы через OpenAI API.
- Сохранение прогресса в `data/save_data.json`.

## Структура

- `src/settings.py` — базовые константы игры (размер окна, физика, пути к ресурсам).
- `src/asset_loader.py` — загрузка спрайтов, звуков и шрифтов.
- `src/main.py` — точка входа, главный цикл и переключение экранов.
- `src/ui.py` — экраны: главное меню, игра, Game Over, настройки, гардероб, Оракул.
- `src/player.py` — логика птицы (физика, анимация).
- `src/obstacles.py` — трубы с динамическим зазором и скоростью.
- `src/data_manager.py` — сейвы и применение настроек.
- `src/ai_core.py` — общение с OpenAI (через `.env` или настройки).
- `assets/` — спрайты, звуки, шрифты (подробно в `assets.md`).

## Установка

```bash
pip install -r requirements.txt
```

Python 3.11+ (у тебя стоит 3.13, он тоже ок) и `pygame` уже включены в `requirements.txt` вместе с `openai` и `python-dotenv`.

## Настройка OpenAI

1. В корне проекта есть файл `.env` (не коммитится в git).
2. Пропиши в нём свой ключ:

```env
OPENAI_API_KEY=sk-...
```

3. Либо введи ключ прямо в настройках игры (экран Settings) — он будет сохранён в `save_data.json`.

При отсутствии ключа:

- Оракул и Game Over фразы работают в офлайн-режиме (локальные тексты).

## Запуск

Из корня проекта:

```bash
python -m src.main
```

## Управление

- В игре: `SPACE` или ЛКМ — прыжок.
- Главное меню:
  - Кнопки мышью: Играть / Гардероб / Оракул / Настройки.
  - Хоткеи: `SPACE` — играть, `K` — гардероб, `O` — Оракул, `S` — настройки.
- Game Over:
  - Кнопки: Ещё раз / Настройки / В меню.
  - Хоткеи: `SPACE` / ЛКМ — заново.
- В Оракуле: текст + `Enter` для вопроса, `Esc` — выход.

## Прогресс и скины

- После каждой игры:
  - Счёт добавляется в валюту.
  - Обновляется рекорд.
- В гардеробе:
  - Базовый скин `blue` открыт сразу.
  - `red`/`yellow` можно купить за валюту.
  - Выбранный скин влияет на спрайт птицы в игре.
